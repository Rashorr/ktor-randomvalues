Get started

This API generates Random Numbers, Strings and UUIDs for you. Additionally multiple Values can be generated with only one request.

get Random Numbers

Random Numbers within a specific Range can be generated. To get Random Numbers you need to make a GET call to the following url :
http://localhost:8080/


QUERY PARAMETERS

Parameter Type Description
r1 Integer Specify the first range for the Number
r2 Integer Specify the second range for the Number
value Integer Set how many Numbers shoud be generated. If this parameter is omitted, 1 is used as the default value.

The full request to generate a random Number can look like this:
http://localhost:8080/?r1=0&r2=10&value=4

Get Random Strings

Generate Random Strings with a defined charset and length. To get Random Strings you need to make a GET call to the following url :
http://localhost:8080/


QUERY PARAMETERS

Parameter Type Description
length Integer Specify the length of the string
charset String Specify the charset of the String
value Integer Set how many Strings shoud be generated. If this parameter is omitted, 1 is used as the default value.

The full request to generate a random String can look like this:
http://localhost:8080/?length=10&charset=sdkjöhsdöghskjfs&value=4

Get Random UUIDS

To generate Random UUIDs you need to make a Get call to the following url :
http://localhost:8080/UUID


QUERY PARAMETERS

Parameter Type Description
Value Integer Set how many UUIDs shoud be generated. If this parameter is omitted, 1 is used as the default value.

The full request to generate a random UUID can look like this:
http://localhost:8080/UUID?value=10
